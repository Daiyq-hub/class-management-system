反射机制测试

反射机制说明

反射机制是指程序在运行时可以动态地调用类的方法、访问类的属性，而无需在编译时知道这些方法和属性的名称。 在本系统中，我们使用反射机制实现了通过字符串作为函数名来调用对应方法的功能。 

// 反射机制示例代码
public String invokeMethod(String methodName, String name) throws Exception {
    // 获取方法
    Method method = this.getClass().getMethod(methodName, String.class);
    // 调用方法
    return (String) method.invoke(this, name);
}

// 被调用的方法
public String addStudent(String name) {
    return "通过反射机制添加学生: " + name;
}

映射功能测试

映射功能说明

映射（Map）是一种键值对集合，它提供了快速的查找能力。在本系统中，我们使用映射来存储和管理各种关联数据， 如学生ID与学生信息的映射、用户名与密码的映射等。 

// 映射功能示例代码
// 创建学生ID到学生信息的映射
Map<String, Student> studentMap = new HashMap<>();

// 添加映射关系
studentMap.put("2023001", new Student("张三", "13800138001"));
studentMap.put("2023002", new Student("李四", "13900139002"));

// 通过ID查找学生
Student student = studentMap.get("2023001");
System.out.println("找到学生: " + student.getName());
